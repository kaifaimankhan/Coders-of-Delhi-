Project Overview

A social network data analysis project that cleans user data and generates friend and page recommendations.

Features
Data Cleaning
Remove duplicate friends.
Remove inactive users.
Remove users with missing names.
Remove duplicate liked pages.
Recommendation System
People You May Know

Suggests users based on mutual connections.

Pages You Might Like

Recommends pages based on shared interests.
